fia-arbeidsgiver
================

Api for å kommunisere IA-samarbeidsstatus mellom fagsystemet Fia og innloggede arbeidsgivere.

# Komme i gang

Koden kan enn så lenge ikke kjøres lokalt.

Tester kjøres med (trenger en kjørende docker runtime):

`./gradlew test`
